# Project Overview

This project builds a **Neural Network model** to predict whether a person has diabetes based on medical features.

The model is trained on the **PIMA Indians Diabetes Dataset** and includes data preprocessing, visualization, and performance evaluation.

---

# Features Used

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age


# Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* TensorFlow / Keras


# Workflow

# 1. Data Preprocessing

* Checked missing values
* Feature scaling using StandardScaler
* Train-test split

# 2. Data Visualization

* Correlation heatmap
* Feature distributions
* Scatter plots

# 3. Model Building

* Dense Neural Network
* ReLU activation for hidden layers
* Sigmoid activation for output

### 4. Model Training

* Optimizer: Adam
* Loss: Binary Crossentropy
* Evaluation using accuracy
